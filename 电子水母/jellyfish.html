数学公式生成的电子水母
"原来数字的世界这么美"
电子水母生成公式
k = 5 cos(x/14) cos(y/30)
e = y/8 - 13
d = (k² + e²)/59 + 4
a = arctan(k, e)
q = 60 - sin(a·e) + k·(3 + (4/d)·sin(d² - 2t))
c = d/2 + e/99 - t/18
X = q · sin(c) + 200
Y = (q + 9d) · cos(c) + 200
// t = 时间变量，实现动画效果

这个电子水母完全由数学公式生成，没有任何预定义的图形或图像。

通过参数方程将平面上的点(x, y)映射到屏幕坐标(X, Y)，并随时间t变化产生动画效果。

调整控制面板中的参数可以改变水母的行为和外观，体验数学如何创造生命般的动态美感。
